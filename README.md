## Introduction

This project helps to use [Linstor Controller and Satellite](https://linbit.com/drbd-user-guide/linstor-guide-1_0-en/)
in Docker containers for Debian, Ubuntu and Alpine.

It is a base image for [Linstor Controller Docker Image](https://github.com/EdmonDantes/linstor-controller-docker)
and [Linstor Satellite Docker Image](https://github.com/EdmonDantes/linstor-satellite-docker)

## Usage

Please take a look at [Linstor Controller Docker Image](https://github.com/EdmonDantes/linstor-controller-docker)
and/or [Linstor Satellite Docker Image](https://github.com/EdmonDantes/linstor-satellite-docker) for more information
